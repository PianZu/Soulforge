# Soulforge

**Soulforge** ist ein kleines 2D-Roguelike, das ich aus Spaß in meiner Freizeit programmiere.  
Das Projekt dient mir hauptsächlich dazu, neue Technologien auszuprobieren und meine Fähigkeiten in Game Development und Webentwicklung zu verbessern.

## Tech-Stack

### Backend
- **Java**

### Frontend
- **Vite**
- **React**
- **Phaser**
- **TypeScript**

## Ziel des Projekts

- Ein 2D-Roguelike mit zufällig generierten Runs
- Praxis in:
  - Game-Loop-Logik mit Phaser
  - State-Management im Frontend
  - Zusammenspiel von Java-Backend und modernem Web-Frontend
  - TypeScript-Typisierung in einem größeren Projekt

> Dieses Projekt ist rein hobbymäßig und nicht kommerziell gedacht.
